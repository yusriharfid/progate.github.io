Tentang Kami

J20 adalah program beasiswa pelatihan intensif bagi 60.000 peserta yang bertujuan untuk meningkatkan keterampilan dan daya saing SDM bidang teknologi informasi dan komunikasi sebagai bagian dari program pembangunan prioritas nasional.

Program J20 secara garis besar dibagi menjadi empat akademi, yaitu: Android Developer, HTML, CSS dan Javascript, iOS Developer, dan Associate Cloud Engineer.
